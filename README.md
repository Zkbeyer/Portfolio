# Zack Beyer — Portfolio Website

Personal portfolio website showcasing my software engineering projects, technical skills, and experience. Built to serve as a central hub for my work, experiments, and featured builds.

---

## Overview

This site highlights:
- Featured software projects
- Technical stacks and tools used
- Screenshots and project summaries
- Direct links to repositories and live demos
- Contact and background information

It is designed to be clean, fast, and project-focused — built primarily to help recruiters and collaborators quickly understand what I build and how I think.

---

## Tech Stack

- React
- TypeScript
- Vite
- Node.js
- Modern CSS
- Static asset hosting

---

## Features

- Project card system with expandable detail overlays
- Structured project metadata (role, timeframe, status, tech stack)
- Multi-image project galleries
- GitHub + live demo linking
- Responsive layout
- Typed project data model
- Easy project entry system via `projectsData.ts`
